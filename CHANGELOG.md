## [0.1.0] — 2026-02-24

### Додано

- Структура репозиторію: `data/`, `notebooks/`, `src/`, `reports/figures/`
- `.gitignore` для Python, Jupyter, venv, `data/raw/`
- README з метою проєкту, джерелом даних та трьома гіпотезами для аналізу
- Скрипт завантаження даних: `src/load_data.py` (реєстр закладів ВО з data.gov.ua)
- Перевірка якості даних: `src/data_quality.py` (пропуски, дублікати, узгодженість регіонів)
- Дослідницький аналіз: `src/data_research.py` (розподіли за регіоном, фінансуванням, десятиліттям; агрегації)
- Візуалізації: `src/visualization.py` — графіки в `reports/figures/` (топ регіонів, тип фінансування, реєстрація по десятиліттям)
- Секція «Що зроблено» у README

### Джерело даних

- Реєстр суб'єктів освітньої діяльності (data.gov.ua)
