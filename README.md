## Питання / гіпотези для аналізу

1. **Регіональний розподіл.** Більшість закладів вищої освіти зосереджена в кількох регіонах (наприклад, м. Київ, Харківська, Львівська, Одеська області), тоді як в інших регіонах їх значно менше.

2. **Державні та приватні заклади.** Співвідношення державних і приватних закладів відрізняється за регіонами; приватні заклади частіше з’являлися в 1990–2000-х роках (пік реєстрації).

3. **Типи закладів і рік реєстрації.** Старіші заклади (нижчий `registration_year`) переважно мають тип «Університет» або «Академія» і державне фінансування; новіші — частіше інститути або філії, у тому числі приватні.
